Enum 

fixed set of constant values, called Enum constants
it is like a class, but specififed group of named constants
Enums can have fields, constructors, and methods
Enum constants are typicallt written in uppercase
using 
Level.LOW  to refer the constants


enum Level {
    LOW,
    MEDIUM,
    HIGH
}


public class TestEnum {
    enum Level {
        LOW, MEDIUM, HIGH
    }

    public static void main(String[] args) {
        Level currentLevel = Level.MEDIUM;
        System.out.println(currentLevel);


// Using enum in switch
        switch(currentLevel) {
            case LOW:
                System.out.println("Low level");
                break;
            case MEDIUM:
                System.out.println("Medium level");
                break;
            case HIGH:
                System.out.println("High level");
                break;
        }
    }
}

-------------------------
Enums can have fields, constructors, and methods

enum Size {
    SMALL(1), MEDIUM(2), LARGE(3);

    private final int value;

    Size(int value) {
        this.value = value;
    }

    public int getValue() {
        return value;
    }
}
-------------------------

values(): Returns an array of all enum constants.

ordinal(): Returns the position of the constant (starting from 0).

valueOf(String name): Returns the enum constant with the specified name.

name(): Returns the name of the enum constant as a string.
