swagger 

an open src framework used for documenting and testing RESTful apis
this will generate interactive api documentation in human readable and machine readble
to explore api endpoints, understand requet and response formats, even perform api testing 
tjrough a user freidny web interface,

add the SpringDoc OpenAPI dependency to your Maven pom.xml

<dependency>
   <groupId>org.springdoc</groupId>
   <artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
   <version>2.8.6</version>
</dependency>

2.application.properties, you can configure the API docs path:


springdoc.api-docs.path=/api-docs

3.Access the Swagger UI in your browser

http://localhost:8080/swagger-ui/index.html

raw open api json is on 
http://localhost:8080/api-docs

Swagger UI and the OpenAPI documentation run on the same port as your Spring Boot application. If your Spring Boot app is running on port 8080
